Yah UI (User Interface) components ko develop karne mein madad karta hai. React ka mukhya uddeshya ek powerful aur interactive user interface banane mein madad karna hai. Isme reusable aur maintainable code likhne ki suvidha hoti hai, jo developers ke liye kaafi faydemand hai. React virtual DOM ka istemal karta hai jo performance ko badhata hai, aur iska use single page applications (SPA) ke develop karne mein hota hai. b.Ajax- Ajax (Asynchronous JavaScript and XML) ek technique hai jo web pages ko dynamic aur interactive banane mein madad karta hai. Isse web pages ko reload kiye bina server se data exchange karne ki suvidha hoti hai. Jab user koi action lete hai jaise button click ya form submit, tab Ajax se background mein server se data lekar aata hai aur page ko update karta hai. Isse user experience improve hota hai kyunki page reload nahi hota aur data turant dikhaya ja sakta hai. Yeh technique JavaScript ka use karti hai aur modern web development mein kaafi popular hai. c.Api- API (Application Programming Interface) ek tarah ka set hota hai jo ek software application ya service ke functions aur features ko dusre applications se interact karne ke liye allow karta hai. Ye interactions hoti hai data exchange, commands, aur requests ke roop mein. APIs ka upyog hota hai jab ek application dusri application se kuch maangti hai ya information share karti hai. Example- Jaise maan lo aap ek restaurant ke website par hote hain aur wahan se menu dekhna chahte hain. Jab aap menu page open karte hain, website background mein API se connect hoti hai jo restaurant ke servers se menu ki information fetch karti hai aur aapko dikhati hai. Is tarah se API aapke browser aur restaurant ke servers ke beech communication ko allow karti hai, jo aapko menu dekhne ki suvidha deti hai bina directly restaurant se baat kiye. APIs alag-alag types ke hote hain jaise RESTful APIs, JSON APIs, SOAP APIs, etc. Inka use web development, mobile applications, cloud services, aur bahut se aur fields mein hota hai jahan data exchange aur communication ki zaroorat hoti hai. Rest Api- RESTful API ek approach hai jisme web services ko design karte waqt REST principles ka istemal kiya jata hai. Ye APIs stateless hoti hain, jisme HTTP methods (GET, POST, PUT, DELETE) ka istemal hota hai resources ko access aur manipulate karne ke liye. Har resource ka ek unique URL hota hai. Example: Maan lo aap ek social media app use kar rahe hain. Agar aap kisi user ka profile dekhna chahte hain, toh aap RESTful API se request bhejte hain, jaise: GET /users/123 Server phir response mein user ka data JSON format mein bhejega: Example- { "id": 123, "name": "Alice", "age": 25, "email": "alice@example.com" } Isme "users" resource ke liye URL /users/123 hai, aur GET request se server ne user "Alice" ka data bheja. Ye hai RESTful API ka example, jisme resources ko unique URLs se access kiya jata hai. e.Json- JSON (JavaScript Object Notation) ek lightweight data interchange format hai jo data ko store aur exchange karne ke liye istemal hota hai. Ye human-readable hai aur easy to parse hai, isliye web applications, APIs, aur programming languages mein popular hai. JSON mein data key-value pairs mein hota hai jise objects ke roop mein represent kiya jata hai. Example:- { "name": "John Doe", "age": 30, "city": "New York", "email": "johndoe@example.com" } Is JSON example mein "name", "age", "city", aur "email" keys hain, aur inke corresponding values "John Doe", 30, "New York", aur "johndoe@example.com" hain. Ye format data ko easily organize aur transport karne mein madad karta hai. Axios- Axios ek popular JavaScript library hai jo client-side mein HTTP requests bhejne aur receive karne mein madad karta hai. Ye modern web applications mein data fetch karne aur server se interact karne ke liye istemal hota hai. Axios easy-to-use syntax provide karta hai jo developers ko HTTP requests (GET, POST, PUT, DELETE) bhejne mein help karta hai. Iska istemal APIs se data fetch karne, form data submit karne, ya server se data send/receive karne ke liye hota hai. Axios ka use promise-based approach ke saath hota hai, jisse asynchronous operations handle karne mein asani hoti hai.
